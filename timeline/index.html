Timeline

Fortran - 1957

The very first high-level programming language, designed for scientific and engineering calculations.

COBOL - 1958

A language created for business and administrative purposes. It was commonly used in financial applications.

BASIC - 1964

Designed to be simple and easy for beginners to learn. It was included with many computers at the time and significantly influenced personal computing.

C - 1970

Provided low-level access to memory whist maintaining high-level features. Impacted nearly all future imperative languages and is still widely used today.

Smalltalk - 1972

Introduced object-oriented programming and was influential in develping GUIs.

C++ - 1983

A successor to C that introduced object-oriented features and an extended standard library. It's also widely used today and is more popular than C.

Python - 1995

Emphasizes readability and simplicity, commonly used for machine learning and data analysis, but is capable of nearly anything, making it the most popular programming language as of 2024.

Java - 1995

An object oriented language that is designed to be portable and secure. Often used in enterprise and mobile applications.

JavaScript - 1995

Created for the programming of web pages, allowing for interactive and dynamic elements.

PHP - 1995

Mainly used for server-side web development. Supported millions of websites at its prime and is still occasionally used today.

C# - 2000

An object-oriented language developed by Microsoft to compete with Java. Used for enterprise applications and game development in the Unity game engine.

Rust - 2010

A contemporary language built for performance and safety. Mainly used for systems programming.
